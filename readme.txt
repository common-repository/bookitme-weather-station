Bookitme Weather Station id FREE services which bring the most easy way to include hourly weather data directly into Wordpress page and wordpress widgets.
== Description ==

The **Bookitme Weather Station** lets you add weather forecast directly into your page and widgets  
Simply adding a shortcode **[bookitme-weather]** **[bookitme-mobile-weather]** **[bookitme-weather-json]** and  to a page. 
You can set VIDEO or IMAGE for weather forecast background. Parameters are available for setup in the plugin ADMIN interface.
See the Plugin screenshots for examples.


**Sent email into weather@bookitme.com for your API with City and Country name**

**NEW WEATHER API WILL BE GENERATED DEMAND ON YOUR REQUEST from email weather@bookitme.com**

More information about BOOKITME WEATHER STATION API and integration to your web page you can find at <a href="http://www.bookitme.com/weather/" target="_blank">http://www.bookitme.com/weather/</a>

DIRECT LINK BOOKITME WEATHER STATION

http://bookitme.com/weather/wtapi.php?api=[your api]&lab=[your label]

This gives your visitors an efficient way to view hourly weather forecast in the nice user friendly form. 
Shortodes:
**[bookitme-weather]** – display weather forecast on the page or post 

**[bookitme-mobile-weather]** - display forecast optimized for mobile devices

**[bookitme-weather-json]** - plugin return JSON weather forecast data for your next work



**Admin Page:** Setup API METRIC system and weather background image. 
If you not specified input parameters in shortcode definition default values from Admin panel will be used.

== Installation ==
Instructions for installing Bookitme Plugin.
1. Download and extract the Plugin zip file.
2. Upload the folder containing the Plugin files to your WordPress Plugins folder (usually ../wp-content/plugins/ folder).
3. Activate the Plugin via the 'Plugins' menu in WordPress.
4. Add [bookitme-weather] or [bookitme-weather-json] shortcode to a page to display weather forecast on your site.
The Plugin can also be installed directly from the main WordPress Plugin page.

1. Go to the Plugins => Add New page.
2. Enter 'Bookitme'  in the textbox and click the 'Search Plugins' button.
3. In the list of relevant Plugins click the 'Install' link for Simple Sitemap on the right hand side of the page.
4. Click the 'Install Now' button on the popup page.
5. Click 'Activate Plugin' to finish installation.
6. Add [bookitme-weather] shortcode to a page to display the sitemap on your site.
